# selected-projects

Source for https://pianohouseproject.org — the piano house project, this site. One project inside the Piano House Project, the umbrella for all of Nico's work. Built with Next.js (App Router) and MDX.

## Develop

```bash
npm run dev      # http://localhost:3000
npm run check    # validate content/ (frontmatter, image refs, heroes)
npm run build
```

## Editing copy

All visible text lives in `content/`; see `docs/editing.md` for the map. Project, workflow, and deploy notes are in `AGENTS.md`.
